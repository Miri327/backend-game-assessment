# Evaluation-task
Game System Assessment
How to run:

Make sure you have Docker installed.

Run the following command in the root folder:
docker-compose up --build

The system will automatically:

Set up a PostgreSQL database.

Run Prisma migrations.

Execute the main.js script to demonstrate the logic.

הנחיות להפעלת הפרויקט:
ודאו שתוכנת Docker מותקנת ופעילה במחשבכם.

הריצו את הפקודה הבאה בטרמינל מתוך תיקיית השורש של הפרויקט:
docker-compose up --build

המערכת תבצע באופן אוטומטי את הצעדים הבאים:

הקמה והגדרה של מסד נתונים PostgreSQL.

סנכרון מבנה הנתונים (Prisma Migrations) מול בסיס הנתונים.

הרצת סקריפט ה-main.js לצורך הדגמת הלוגיקה העסקית והצגת התוצאות ב-Logs.
